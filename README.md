# Ledger Rucoy

## Deskripsi
Ledger Rucoy adalah aplikasi web untuk mencatat transaksi keuangan dengan tampilan modern dan ringan. Aplikasi ini memungkinkan pengguna untuk menambahkan, mengelola, dan mengekspor catatan transaksi mereka dengan mudah.

## Struktur Proyek
Proyek ini terdiri dari beberapa file dan folder:

- `index.html`: Struktur utama aplikasi web yang mencakup elemen HTML yang diperlukan, tautan ke file CSS dan JavaScript, serta tata letak untuk catatan transaksi.
- `css/styles.css`: File CSS yang mendefinisikan penampilan visual aplikasi, termasuk warna, font, tata letak, dan desain responsif.
- `js/main.js`: File JavaScript yang mengelola fungsionalitas aplikasi, termasuk penanganan pengiriman formulir, rendering entri transaksi, dan pengelolaan penyimpanan lokal.

## Instalasi
1. Clone repositori ini ke mesin lokal Anda.
2. Buka file `index.html` di browser untuk menjalankan aplikasi.

## Penggunaan
- Tambahkan transaksi baru dengan mengisi formulir di sisi kiri aplikasi.
- Gunakan tombol filter untuk melihat transaksi berdasarkan jenis (BUY/SELL).
- Ekspor catatan transaksi ke dalam format CSV dengan mengklik tombol "Export CSV".
